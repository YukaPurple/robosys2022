# robosys2022
ロボットシステム学2022

# plus.py
- 標準入力から読み込んだ数字の和を出力します。
- 実行例

```
$ seq 5 | ./plus.py
		
15.0
```

# times.py
- 標準入力から読み込んだ数字の積を出力します。
- 実行例

```	
$ seq 5 | ./times.py

120.0
```
